MANAGEMENT MORALE ENHANCEMENT SIMULATOR

Welcome, Middle Manager #4281. This tool will help you maximize IT department morale through strategic pizza deployment instead of actual compensation.

Your objective: Reach 500 MANAGER POINTS to earn your promotion while maintaining morale above CRITICAL levels.

HOW TO PLAY:

  • Click the pizza to earn Pizzas (the 3D pizza on the left side)
  • Pizzas automatically generate over time
  • Use Pizzas to purchase store upgrades
  • Each upgrade has a cooldown period before it can be used again
  • Upgrades become more expensive each time you use them
  • Random events will decrease morale - react quickly!

STORE UPGRADES:

  • Display Stand - Change pizza rotation speed to mesmerize staff (+10 MANAGER POINTS, +10% pizza production)
  • Topping Station - Add a new topping to distract developers (+25 MANAGER POINTS, +50% pizza production)
  • Pizza Oven - Install a new oven to boost morale (+50 MANAGER POINTS, +100% pizza production)
  • Delivery Truck - Allocate more budget to pizza, less to salaries (+100 MANAGER POINTS, +400% pizza production)

WARNING: Random events like deadlines, bugs, and competitor job offers will occur. React quickly to maintain morale!

REMEMBER: Pizza is NOT a substitute for competitive compensation, but don't tell the developers that.
